\doxysection*{Dependencies and instructions}

In order to execute the program, the user has to do the following\+: 
\begin{DoxyItemize}
\item make sure the datasets are in the \char`\"{}res\char`\"{} folder 
\item if a new dataset is created, must have the same name of the original~\newline
 datasets or the name on the \char`\"{}main.\+cpp\char`\"{} and \char`\"{}scenario3.\+cpp\char`\"{} files~\newline
 must be changed  
\item if more lines are added to the dataset files, make sure there are no~\newline
 extra empty lines 
\end{DoxyItemize}
